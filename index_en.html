Usage

1.Find the Firefox profiles folder (if you do not know where it is, you can open about: support, click the "open folder" on right of "configuration folder")

2.Copy out stylish.sqlite

3.Select stylish.sqlite below

4.Wait a moment, the browser will pop up the download window, save it as

5.Open the xStyle management interface, select "Retrieve Backup", select the file just downloaded

Note: This will overwrite the existing style in xStyle
